# Configuração basica do Adonis

Para criar o projeto você precisa rodar o seguinte comando : 

> npm init adonisjs@latest aula3 nomedoprojeto

Durante a criação vai perguntar se deseja instalar as devidas depedencias do NodeJS, aonde vamos apertar o Y+ENTER.

Após esse processo, devemos selecionar a opção hypermedia app e apertar 'ENTER'

O node vai finalizar a instalação e na sequência, precisamos executar com esses passos:

'''powershell

    # 1. Passo é entrar na pasta do projeto usando o comando:
    cd nomedoprojeto

    # 2. Passo é executar usando o comando:
    npm run dev

    # 3. Entrar na página http://localhost:porta (porta é o numero que ele gera, normalmente é 3333)

    # 4. Passo opcional mas recomendado, é abrir o vscode dentro do projeto com o comando:
    code .
 ```
Para editar o vizualizar (HTML) precisamos ir na pasta `resources\views\pages` e editar o arquivo `home.edge` que é o nosso HTML
